# Market Discord Bot

Discord の Modal を使って、プライベートマーケット用のアカウント作成・承認フローを行う Bot です。

この Bot は、Web アプリ側で直接アカウント登録をさせるのではなく、Discord サーバー内で `/setup` を実行してアカウント申請を行うためのものです。

## 概要

この Bot は以下の処理を担当します。

- `/setup` コマンドでアカウント作成 Modal を表示
- Discord ユーザー ID を使ってアカウントを作成
- 初期パスワードを自動生成
- パスワードをハッシュ化して DB に保存
- 初期パスワードをユーザーの DM に送信
- 管理者チャンネルにアカウント申請通知を送信
- Approve / Reject ボタンで申請を承認・拒否
- 承認時に Discord ロールを付与
- `/profile` で登録状態を確認

## 想定している用途

身内・学校・コミュニティ内だけで使うプライベートフリーマーケット向けです。

Web アプリ側では通常のサインアップを用意せず、Discord 上でアカウント作成を行います。

/setup
↓
